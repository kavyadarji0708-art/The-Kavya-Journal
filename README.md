# The Kavya Journal

A modern essay publishing platform designed to give young writers a place to share thoughtful, high-quality writing while encouraging meaningful discussion and critical thinking.

## Features

### For Authors
- Create an account and build a portfolio
- Submit essays across a variety of topics
- Track submission status through the review process
- Receive feedback and revision requests from editors
- Publish to a community audience

### For Readers
- Browse essays by category
- Search for topics of interest
- Interact with authors and other readers
- Discover diverse perspectives
- Access a clean, accessible reading experience

### For Administrators
- Review submitted essays in a dedicated dashboard
- Accept, decline, or request revisions
- Manage categories and platform content
- Monitor community standards
- Ensure editorial quality

## Tech Stack

- **Frontend:** React, TypeScript, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** PostgreSQL
- **Authentication:** JWT
- **Deployment:** Docker, AWS/Vercel

## Project Structure

```

├── database/              # PostgreSQL schemas and migrations
├── docker-compose.yml     # Local development setup
```
